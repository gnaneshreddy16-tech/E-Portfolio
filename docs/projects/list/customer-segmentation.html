Overview

This project applies K-means clustering to segment retail customers based on purchase frequency, recency, and spend — a classic RFM (Recency, Frequency, Monetary) analysis used widely in business intelligence and marketing strategy.

NoteDataset

Synthetically generated customer transaction data modelled on realistic retail distributions. The same pipeline applies directly to real CRM exports.

What is RFM Analysis?

Metric Definition Business Meaning
Recency Days since last purchase How recently did they buy?
Frequency Number of purchases How often do they buy?
Monetary Total spend How much do they spend?

Data Generation & Clustering

Code 
import pandas as pd
import numpy as np
import plotly.express as px
from sklearn.cluster import KMeans
from sklearn.preprocessing import StandardScaler

np.random.seed(21)
n = 200

df = pd.DataFrame({
    'CustomerID': range(1, n+1),
    'Recency':    np.random.randint(1, 365, n),
    'Frequency':  np.random.randint(1, 50, n),
    'Monetary':   np.random.randint(100, 5000, n)
})

scaler = StandardScaler()
scaled = scaler.fit_transform(df[['Recency', 'Frequency', 'Monetary']])

kmeans = KMeans(n_clusters=4, random_state=42, n_init=10)
df['Segment'] = kmeans.fit_predict(scaled)

segment_names = {0: 'High Value', 1: 'At Risk', 2: 'Loyal', 3: 'New Customers'}
df['Segment'] = df['Segment'].map(segment_names)

print("Segment Averages:")
print(df.groupby('Segment')[['Recency', 'Frequency', 'Monetary']].mean().round(1))

High Value customers spend on average 4x more than New Customers — prioritising retention efforts is critical.

## Overview

This project applies K-means clustering to segment retail customers based on purchase frequency, recency, and spend — a classic RFM (Recency, Frequency, Monetary) analysis used widely in business intelligence and marketing strategy.

## What is RFM Analysis?

| Metric | Definition | Business Meaning |

| **Recency** | Days since last purchase | How recently did they buy? |

## Data Generation & Clustering

```{python}
#| echo: true
#| code-fold: true
#| warning: false
#| label: clustering

import pandas as pd
import numpy as np
import plotly.express as px
from sklearn.cluster import KMeans
from sklearn.preprocessing import StandardScaler

np.random.seed(21)
n = 200

df = pd.DataFrame({
    'CustomerID': range(1, n+1),
    'Recency':    np.random.randint(1, 365, n),
    'Frequency':  np.random.randint(1, 50, n),
    'Monetary':   np.random.randint(100, 5000, n)
})

scaler = StandardScaler()
scaled = scaler.fit_transform(df[['Recency', 'Frequency', 'Monetary']])

kmeans = KMeans(n_clusters=4, random_state=42, n_init=10)
df['Segment'] = kmeans.fit_predict(scaled)

segment_names = {0: 'High Value', 1: 'At Risk', 2: 'Loyal', 3: 'New Customers'}
df['Segment'] = df['Segment'].map(segment_names)

print("Segment Averages:")
print(df.groupby('Segment')[['Recency', 'Frequency', 'Monetary']].mean().round(1))
```
