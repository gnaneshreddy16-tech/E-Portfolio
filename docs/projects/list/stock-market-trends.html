Overview

This project analyses simulated S&P 500 sector performance data to identify trends, compare volatility across sectors, and visualise cumulative returns over a 12-month period. The goal is to demonstrate how Python can replace manual spreadsheet analysis for financial decision-making.

NoteDataset

This project uses synthetically generated data modelled on realistic sector return distributions. In a production setting, this pipeline would connect to a financial API such as Yahoo Finance or Alpha Vantage.

Setup and Data Generation

Code 
import pandas as pd
import numpy as np
import plotly.express as px
import plotly.graph_objects as go
from plotly.subplots import make_subplots

np.random.seed(42)

sectors = ['Technology', 'Finance', 'Healthcare', 'Energy', 'Consumer']
months = pd.date_range(start='2024-01-01', periods=12, freq='MS')

# Simulate monthly returns (percent)
returns_data = {
    'Technology': [3.2, -1.1, 4.5, 2.8, -0.9, 5.1, 1.3, 3.7, -2.1, 4.2, 2.9, 6.1],
    'Finance':    [1.8, 0.5, 2.1, -0.7, 1.9, 2.4, 0.8, 1.5, 2.2, -1.1, 3.1, 1.7],
    'Healthcare': [0.9, 1.4, -0.3, 2.1, 1.1, 0.7, 2.3, 1.8, 0.5, 2.6, 1.2, 1.9],
    'Energy':     [-2.1, 3.4, 1.7, -1.2, 4.1, -0.8, 2.9, -1.5, 3.2, 1.1, -0.6, 2.3],
    'Consumer':   [1.1, 0.8, 1.9, 1.3, 0.4, 1.7, 0.9, 2.1, 1.4, 0.6, 2.3, 1.5]
}

df_returns = pd.DataFrame(returns_data, index=months)
df_returns.index.name = 'Date'

print("Monthly Returns Summary:")
print(df_returns.describe().round(2))

Monthly Returns by Sector

Code 
df_melted = df_returns.reset_index().melt(id_vars='Date', var_name='Sector', value_name='Return (%)')

fig = px.line(
    df_melted,
    x='Date',
    y='Return (%)',
    color='Sector',
    markers=True,
    title='Monthly Sector Returns — 2024',
    template='plotly_white',
    color_discrete_sequence=px.colors.qualitative.Set2

fig.update_layout(
    font_family='Source Sans 3',
    title_font_size=18,

    xaxis_title='Month',

    hovermode='x unified'

fig.show()

Cumulative Returns

Code 
df_cumulative = (1 + df_returns / 100).cumprod() - 1
df_cumulative = df_cumulative * 100

df_cum_melted = df_cumulative.reset_index().melt(id_vars='Date', var_name='Sector', value_name='Cumulative Return (%)')

fig2 = px.area(
    df_cum_melted,
    x='Date',
    y='Cumulative Return (%)',
    color='Sector',
    title='Cumulative Sector Returns — 2024',
    template='plotly_white',
    color_discrete_sequence=px.colors.qualitative.Set2
)

fig2.update_layout(
    font_family='Source Sans 3',
    title_font_size=18,
    hovermode='x unified'
)

fig2.show()

Cumulative returns show Technology significantly outperforming other sectors by year-end.

Volatility Comparison

Code 
volatility = df_returns.std().reset_index()
volatility.columns = ['Sector', 'Volatility (std dev)']
mean_return = df_returns.mean().reset_index()
mean_return.columns = ['Sector', 'Mean Monthly Return (%)']

summary = volatility.merge(mean_return, on='Sector')

fig3 = px.scatter(
    summary,
    x='Volatility (std dev)',
    y='Mean Monthly Return (%)',
    text='Sector',
    size='Mean Monthly Return (%)',
    color='Sector',
    title='Risk vs Return by Sector',
    template='plotly_white',
    color_discrete_sequence=px.colors.qualitative.Set2
)

fig3.update_traces(textposition='top center')
fig3.update_layout(
    font_family='Source Sans 3',
    title_font_size=18,
    showlegend=False
)

fig3.show()

Technology shows the highest return but also the highest volatility — a classic risk/return tradeoff.

## Setup and Data Generation

```{python}
#| echo: true
#| code-fold: true
#| label: setup
#| warning: false

import pandas as pd
import numpy as np
import plotly.express as px
import plotly.graph_objects as go
from plotly.subplots import make_subplots

np.random.seed(42)

sectors = ['Technology', 'Finance', 'Healthcare', 'Energy', 'Consumer']
months = pd.date_range(start='2024-01-01', periods=12, freq='MS')

# Simulate monthly returns (percent)
returns_data = {
    'Technology': [3.2, -1.1, 4.5, 2.8, -0.9, 5.1, 1.3, 3.7, -2.1, 4.2, 2.9, 6.1],
    'Finance':    [1.8, 0.5, 2.1, -0.7, 1.9, 2.4, 0.8, 1.5, 2.2, -1.1, 3.1, 1.7],
    'Healthcare': [0.9, 1.4, -0.3, 2.1, 1.1, 0.7, 2.3, 1.8, 0.5, 2.6, 1.2, 1.9],
    'Energy':     [-2.1, 3.4, 1.7, -1.2, 4.1, -0.8, 2.9, -1.5, 3.2, 1.1, -0.6, 2.3],
    'Consumer':   [1.1, 0.8, 1.9, 1.3, 0.4, 1.7, 0.9, 2.1, 1.4, 0.6, 2.3, 1.5]
}

df_returns = pd.DataFrame(returns_data, index=months)
df_returns.index.name = 'Date'

print("Monthly Returns Summary:")
print(df_returns.describe().round(2))
```

## Cumulative Returns

```{python}

df_cumulative = (1 + df_returns / 100).cumprod() - 1

fig2 = px.area(
```

## Volatility Comparison

```{python}
#| echo: true
#| code-fold: true
#| fig-cap: "Technology shows the highest return but also the highest volatility — a classic risk/return tradeoff."
#| label: volatility

volatility = df_returns.std().reset_index()
volatility.columns = ['Sector', 'Volatility (std dev)']
mean_return = df_returns.mean().reset_index()
mean_return.columns = ['Sector', 'Mean Monthly Return (%)']

summary = volatility.merge(mean_return, on='Sector')

fig3 = px.scatter(
    summary,
    x='Volatility (std dev)',
    y='Mean Monthly Return (%)',
    text='Sector',
    size='Mean Monthly Return (%)',
    color='Sector',
    title='Risk vs Return by Sector',
    template='plotly_white',
    color_discrete_sequence=px.colors.qualitative.Set2
)

fig3.update_traces(textposition='top center')
fig3.update_layout(
    font_family='Source Sans 3',
    title_font_size=18,
    showlegend=False
)

fig3.show()
```
