Overview

This project performs a full sales analytics workflow on a simulated retail dataset covering 12 months of transactions across four product categories and three regions. The analysis identifies top-performing segments, seasonal patterns, and provides a simple revenue forecast.

NoteDataset

Synthetically generated to reflect realistic retail sales distributions. The same analytical approach applies directly to real CRM or ERP exports.

Data Generation

Code 
import pandas as pd
import numpy as np
import plotly.express as px
import plotly.graph_objects as go

np.random.seed(7)

months = pd.date_range(start='2024-01-01', periods=12, freq='MS')
categories = ['Electronics', 'Clothing', 'Home & Garden', 'Food & Beverage']
regions = ['Dublin', 'Cork', 'Galway']

rows = []
for month in months:
    for cat in categories:
        for region in regions:
            base = {'Electronics': 45000, 'Clothing': 28000,
                    'Home & Garden': 18000, 'Food & Beverage': 32000}[cat]
            seasonal = 1 + 0.15 * np.sin((month.month - 3) * np.pi / 6)
            noise = np.random.uniform(0.88, 1.12)
            revenue = round(base * seasonal * noise, 2)
            units = int(revenue / np.random.uniform(25, 85))
            rows.append({
                'Month': month,
                'Category': cat,
                'Region': region,
                'Revenue': revenue,
                'Units': units
            })

df = pd.DataFrame(rows)
print(f"Dataset: {len(df)} records")
print(df.groupby('Category')['Revenue'].sum().sort_values(ascending=False).apply(lambda x: f"€{x:,.0f}"))

Monthly Revenue Trend

Code 
monthly = df.groupby('Month')['Revenue'].sum().reset_index()

fig = px.bar(
    monthly,
    x='Month',
    y='Revenue',
    title='Total Monthly Revenue — 2024',
    template='plotly_white',
    color='Revenue',

    text_auto='.2s'

fig.update_layout(

fig.show()

Simple Revenue Forecast (Next 3 Months)

Code 
monthly['Rolling_Avg'] = monthly['Revenue'].rolling(3).mean()

forecast_months = pd.date_range(start='2025-01-01', periods=3, freq='MS')
last_avg = monthly['Rolling_Avg'].iloc[-1]
growth_rate = 0.02
forecast_values = [last_avg * (1 + growth_rate) ** i for i in range(1, 4)]

forecast_df = pd.DataFrame({'Month': forecast_months, 'Revenue': forecast_values})
forecast_df['Type'] = 'Forecast'
monthly['Type'] = 'Actual'

combined = pd.concat([monthly[['Month', 'Revenue', 'Type']], forecast_df])

fig4 = px.line(
    combined,
    x='Month',
    y='Revenue',
    color='Type',
    markers=True,
    title='Revenue Trend with 3-Month Forecast',
    template='plotly_white',
    color_discrete_map={'Actual': '#2980b9', 'Forecast': '#e74c3c'}
)

fig4.update_layout(
    font_family='Source Sans 3',
    title_font_size=18,
    yaxis_tickprefix='€',
    hovermode='x unified'
)

fig4.show()

3-month revenue forecast using rolling average trend — projected Q1 2025 growth of approximately 6%.
